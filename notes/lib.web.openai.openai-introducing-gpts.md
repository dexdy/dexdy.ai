OpenAI introduced GPTs, which are customizable versions of ChatGPT tailored for specific tasks and purposes. Users can create their own GPTs without needing to code, integrating abilities such as web search, image creation, or data analysis. These GPTs can be made for personal use, company use, or public sharing. OpenAI also plans to launch a GPT Store to feature and monetize community-created GPTs. The platform emphasizes user privacy and data control, with safety measures in place to review GPTs against usage policies and allow user reporting of concerns​​.
