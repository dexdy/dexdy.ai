URL :: https://github.com/serokell/deploy-rs

`deploy-rs` is a Rust-based Nix-flake deploy tool that simplifies deploying multiple profiles to specified nodes. It supports flexible deployment options, including magic rollback to prevent system unavailability. The tool is designed for Nix flakes, enabling independent, lesser-privileged profile deployments. The project is maintained by Serokell, and users can seek support on Matrix.
