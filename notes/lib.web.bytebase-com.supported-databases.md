URL :: https://www.bytebase.com/docs/introduction/supported-databases/

The page on Bytebase provides a list of supported databases and their
respective versions. These include MySQL, PostgreSQL, TiDB, ClickHouse,
Snowflake, Redshift, MongoDB, Spanner, Redis, Oracle, Microsoft SQL Server, MariaDB, OceanBase, and RisingWave. Each database has specific version requirements. Additionally, the page offers a detailed feature matrix for different database functionalities such as Change Workflow, Advanced Schema Management, SQL Editor, Data Security & Governance, Disaster Recovery, and Anomaly Detection, indicating which features are supported for each database.
