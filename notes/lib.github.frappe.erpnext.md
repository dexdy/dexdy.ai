URL :: https://github.com/frappe/erpnext

ERPNext is an open-source Enterprise Resource Planning (ERP) software developed by Frappe Technologies. It's built on the Frappe Framework, a full-stack web app framework using Python & JavaScript. ERPNext includes modules for accounting, warehouse management, CRM, sales, purchase, HRMS, project management, support, asset management, quality management, manufacturing, website management, and more. The software is licensed under the GNU General Public License (v3), and its documentation is available under the Creative Commons license. ERPNext can be installed via Docker or manually, and it supports a range of business functions and industries.
