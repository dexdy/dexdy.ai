**Mamba: a new sequence model without attention**

  * Mamba is a neural network architecture that uses selective structured state space models (SSMs) instead of attention or MLP blocks to process long sequences efficiently and effectively.
  * Mamba can perform content-based reasoning by letting the SSM parameters depend on the input tokens, and can run in parallel using a hardware-aware algorithm.
  * Mamba achieves state-of-the-art performance on several modalities such as language, audio, and genomics, and outperforms or matches Transformers of the same or larger size.
