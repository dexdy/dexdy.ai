URL :: https://github.com/jaegertracing

The GitHub page for Jaeger, a Cloud Native Computing Foundation (CNCF) project, showcases its role as a distributed tracing platform. It provides a range of repositories related to the Jaeger Distributed Tracing project, including its core software, documentation, user interface, data model definitions, Kubernetes operator, and various tools for integration and deployment. These resources are designed to help developers implement and manage Jaeger within their applications and infrastructure, facilitating troubleshooting and monitoring of microservices-based architectures.
