URL :: https://github.com/atuinsh/atuin

The GitHub repository for Atuin describes it as a tool that enhances shell history by using a SQLite database for storing command history with additional context. It offers encrypted synchronization of command histories across different machines through an Atuin server. Atuin supports several shells including zsh, bash, fish, nushell, and xonsh. It provides features like a full-screen history search UI, encrypted history backup and sync, and statistics on command usage. The project is licensed under the MIT license and encourages community contributions and discussions through forums and Discord.
