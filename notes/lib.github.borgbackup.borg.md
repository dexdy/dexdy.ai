URL :: https://github.com/borgbackup/borg

The BorgBackup is a deduplicating archiver with compression and authenticated encryption, written in Python. It is available on GitHub under the borgbackup organization. The latest release is version 1.4.0a1, which is an alpha release for testing purposes. The project also has a website and a service specification. The software is not recommended for use on production backup repositories. You can find the detailed release notes and installation instructions on the BorgBackup's GitHub page.
