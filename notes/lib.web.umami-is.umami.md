URL :: https://umami.is/

Umami is a web analytics tool that aims to simplify the process of collecting, analyzing, and understanding web data while prioritizing visitor privacy and data ownership. It offers simple analytics, collects only the metrics that users care about, and helps maintain compliance with data privacy laws by anonymizing all collected data. Umami does not collect personal information, use cookies, or track users across websites, and it is GDPR compliant. It is a self-hosted, open-source solution that allows for customization of the database and hosting. The tool is designed to be easy to set up and use, and it provides a shareable link to the analytics with one click. The Umami project is available on GitHub, where detailed installation and testing instructions can be found. It also supports SQL and PostgreSQL for databases. Additionally, Umami can be integrated into web applications, and it is used by platforms like Vercel and Hashnode for analytics.
