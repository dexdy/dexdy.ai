URL :: https://www.rsync.net/

The website for rsync.net focuses on providing cloud storage solutions specifically for offsite backup. They offer users an empty UNIX filesystem that can be accessed with any SSH tool. Their platform is built on ZFS, ensuring data security and fault tolerance. Rsync.net supports various protocols like rsync, sftp, scp, borg, rclone, restic, and git-annex, making it versatile for different backup needs. Additionally, they highlight their commitment to privacy and provide information about pricing, support, and regulatory compliance. For more detailed information, please visit their website at [rsync.net](https://www.rsync.net/).
