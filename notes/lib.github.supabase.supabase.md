URL :: https://github.com/supabase/supabase

The GitHub page for Supabase provides an overview of it as an open-source alternative to Firebase, focusing on delivering similar features through enterprise-grade open-source tools. It highlights key features such as hosted Postgres databases, authentication, auto-generated APIs (REST, GraphQL, Realtime), functions (database and edge), file storage, and an AI toolkit. Supabase is designed to offer a Firebase-like developer experience, leveraging existing open-source tools wherever possible, and contributing back to the community by open-sourcing any new tools it develops. For more details, you can visit the [GitHub page](https://github.com/supabase/supabase).
