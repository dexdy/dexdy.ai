Umami is a simple, fast, privacy-focused alternative to Google Analytics. It is a self-hosted web analytics solution that aims to provide a friendlier, privacy-focused alternative to Google Analytics and a free, open-source alternative to paid solutions. It collects only the metrics you care about and respects the privacy of your users[^1][^3]. The Umami software is available on GitHub under the umami-software organization, and its repositories include the main Umami project, the Umami website, and other related components[^1][^2][^3]. The latest release of Umami (v2.8.0) includes new features, interface updates, and various fixes[^4].
