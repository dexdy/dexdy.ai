URL :: https://github.com/informalsystems/cosmos.nix

The page provides the official repository for cosmos.nix, a collection of Nix packages and tools for the Cosmos SDK and related projects. It details how cosmos.nix aims to provide a comprehensive set of Nix packages for building, testing, and deploying Cosmos SDK-based blockchains and applications. The repository includes packages for the Cosmos SDK, Gaia, IBC, and other related projects, and provides tools for managing and deploying Cosmos SDK-based applications. It also offers documentation, examples, and tutorials to help users get started with cosmos.nix and leverage its capabilities to build and deploy blockchain applications on the Cosmos network.
