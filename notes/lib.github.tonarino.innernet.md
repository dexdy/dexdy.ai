"Innernet" is a private network system that uses WireGuard, a modern VPN protocol, as its underlying technology. Hosted on GitHub by the organization tonarino, it has garnered significant attention, indicated by its 4.7k stars and 178 forks. The project is open-source, licensed under the MIT license. The primary programming language used in its development is Rust, constituting 97.1% of the codebase. For more details, you can visit the [GitHub page of Innernet](https://github.com/tonarino/innernet).
