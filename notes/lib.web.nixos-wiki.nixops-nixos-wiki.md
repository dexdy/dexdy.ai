URL :: https://nixos.wiki/wiki/NixOps

NixOps is a tool for deploying NixOS machines in networks or clouds. It
utilizes a declarative approach, specifying the desired configuration of
machines and automating deployment processes like provisioning and service
management. Key features include multi-cloud support, modularity for
separating logical and physical aspects of deployments, and integration with
Nix package manager and NixOS for advanced package and system configuration
management. It supports various deployment targets, including cloud
environments like Amazon EC2, and local targets like VirtualBox. The tool also
manages non-machine cloud resources, such as Amazon S3 buckets.
