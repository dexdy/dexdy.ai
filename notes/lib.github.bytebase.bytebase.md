URL :: https://github.com/bytebase/bytebase

Bytebase is a tool designed for advanced database DevOps and CI/CD, targeting developers, DBAs, and platform engineering teams. It aims to be the GitLab/GitHub for database DevOps. The project is publicly available on GitHub, where it has garnered significant interest, evidenced by a large number of stars and forks. The repository includes various resources and documentation, with a diverse set of topics like MySQL, Redis, PostgreSQL, DevOps, and CI/CD being associated with it. The codebase primarily consists of Go, TypeScript, and Vue.
